### 🔹 Filtering NULL values

```sql
SELECT * FROM customern WHERE NULL = NULL;
-- Returns no rows (comparison with NULL using = does not work)

SELECT * FROM customern WHERE dob IS NULL;
-- Returns rows where dob is NULL

SELECT * FROM customern WHERE dob IS NOT NULL;
-- Returns rows where dob has a value (not NULL)
```

### 🔹 Handling NULL values

```sql
SELECT *, ISNULL(age, 11) FROM customern;
-- Returns all rows; if age is NULL, it shows 11 instead

SELECT *, ISNULL(dob, '2205-01-01') FROM customern;
-- Returns all rows; if dob is NULL, it shows '2205-01-01'

SELECT *, COALESCE(dob, NULL, NULL, '2020-01-01') AS col FROM customern;
-- Returns first non-NULL value → dob if present, else '2020-01-01'

SELECT *, COALESCE(dob, NULL, '2020-01-03', NULL, '2020-01-01') AS col FROM customern;
-- Returns dob if not NULL, else '2020-01-03'
```

---

### 🔹 COUNT and NULL

```sql
SELECT COUNT(age) FROM customern;
-- Returns count of rows where age is NOT NULL

SELECT COUNT(ISNULL(age, 0)) FROM customern;
-- Returns total number of rows (NULLs replaced with 0, so all rows counted)
```

---

### 🔹 AVG and NULL

```sql
SELECT AVG(age) FROM customern;
-- Returns average of non-NULL age values (NULLs ignored)

SELECT AVG(ISNULL(age, 0)) FROM customern;
-- Returns average including NULLs treated as 0 (divides by total rows)
```
